# 爱的天气提醒

一个温馨的天气提醒网站，帮助用户为心爱的人自动发送天气提醒和生日祝福。

## 功能特点

- 🌤️ **天气自动提醒** - 设置城市和时间，每天自动向TA的微信发送天气情况和温馨提示
- 🎂 **生日祝福** - 记录双方生日，系统会在特殊日子自动发送定制化祝福到TA的微信
- 💝 **情感指数分析** - 分析互动频率和内容，生成情感连接强度报告
- 📊 **访问统计** - 详细的网站访问统计和分析功能
- 📱 **响应式设计** - 支持移动端和桌面端完美显示
- 🎨 **现代化界面** - 使用Tailwind CSS构建的美观界面

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: Tailwind CSS
- **图表**: Chart.js
- **图标**: Font Awesome
- **二维码**: QRCode.js
- **微信分享**: 微信JS-SDK

## 本地运行

1. 克隆仓库
```bash
git clone [仓库地址]
cd tongzhi
```

2. 直接打开 `index.html` 文件即可在浏览器中查看

本项目是纯前端项目，可以直接部署到任何静态网站托管服务。
